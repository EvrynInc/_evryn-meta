   - **Mixed entries:** a `mixblock` bash helper: `printf '%s\n\n' "$HEADER"` + `for ln in <routed bullet lines>; do sed -n "${ln}p" <snapshot>; done` + `printf '\n---\n\n'`.
   - Footer: the truncation canary.
   - 🔴 **Bash-backtick trap (I got bitten TWICE this session):** markdown bullets contain backticks. Use `sed` to copy bullet *lines verbatim* (never retype them), and use `printf '%s'` or **single-quoted** heredocs (`<<'EOF'`) for headers/footers — an *unquoted* backtick in printf/heredoc triggers command substitution and corrupts the file. If you `sed -i 's/…/…/'`, verify after; a bad substitution mangled two files this session.
