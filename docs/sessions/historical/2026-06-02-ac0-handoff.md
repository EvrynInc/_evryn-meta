## Mid-session learnings worth keeping

- **`git rm` refuses a modified file** — if you've edited a file and then want to delete it, `git rm -f`. (Bit me on `developer.md`: the first commit kept the modified file instead of deleting; fixed with `-f`.)
- **Pathspec commits (`git commit -- path`) are the safe form amid concurrent team activity** — they commit only the named paths regardless of what else is staged, so you never sweep another agent's work.
- **Subagents DO inherit the parent project's CLAUDE.md** (AC's), confirmed by a diagnostic subagent. They do NOT auto-load the target agent's own CLAUDE.md — name it explicitly.
